# 实践与认识的辩证运动：可视化案例

本项目是一个交互式网页应用，旨在通过可视化的方式，生动阐释马克思主义哲学中“实践与认识的辩证运动”及其螺旋式上升的规律。应用通过两个真实的科技发展案例——中国“长征五号”运载火箭与美国“SpaceX”系列火箭的迭代历程——来具体展示这一哲学原理。

## ✨ 项目特色

*   **动态可视化**: 以螺旋上升的路径，直观展示“实践-认识-再实践”的无限循环与向前发展的过程。
*   **真实案例**: 内置“长征五号”与“SpaceX”两个详细的案例数据，覆盖了从初期探索到成熟应用的关键发展阶段。
*   **交互式探索**:
    *   **播放控制**: 提供自动播放、暂停、逐轮播放和速度调节功能，方便用户按照自己的节奏学习。
    *   **数据切换**: 可一键切换不同的案例，进行对比分析。
    *   **详细信息**: 在动画的每个关键节点，都会弹出卡片展示该阶段的详细信息；右侧面板同步呈现当前周期的完整情况。
*   **纯净技术栈**: 使用原生的 HTML, CSS, 和 JavaScript 构建，无任何外部框架或库的依赖，轻量且易于理解。
*   **响应式设计**: 界面布局能够适应不同尺寸的屏幕，在桌面和移动设备上均有良好体验。
*   **键盘快捷键**: 支持使用 `空格键` 快速开始或暂停动画。

## 🚀 如何使用

该项目无需复杂的环境配置或编译过程。

1.  下载或克隆本项目代码到本地。
2.  使用现代浏览器（如 Chrome, Firefox, Edge 等）直接打开根目录下的 `index.html` 文件即可开始体验。

### 控制说明

*   **自动播放**: 开始或继续播放动画。
*   **自动暂停**: 开启后，动画会在每一轮（一个完整的“实践-认识-飞跃”循环）结束时自动暂停。
*   **下一个**: 在“自动暂停”开启且动画暂停在轮末时可用，点击后将进入下一轮动画。
*   **速度**: 调节动画播放的速度。
*   **切换到...**: 在“长征五号”和“SpaceX”两个案例之间切换。

## 🔧 技术实现

*   **前端**:
    *   **HTML**: `index.html` 负责构建页面的基本骨架和所有UI元素。
    *   **CSS**: `styles.css` 负责应用的全部样式，使用 CSS 变量（Custom Properties）来管理颜色、尺寸等，便于主题定制。
    *   **JavaScript**: `app.js` 是项目的核心，负责：
        *   **数据管理**: 以 `DATASETS` 对象的形式存储所有案例数据。
        *   **几何计算**: 动态计算并生成 SVG 螺旋路径。
        *   **动画驱动**: 基于 `requestAnimationFrame` 实现平滑的动画循环。
        *   **DOM 操作与事件处理**: 响应用户交互，更新UI状态和信息展示。

## 💡 哲学背景

本项目旨在图解“实践和认识的辩证关系原理”，其核心思想包括：

1.  **实践决定认识**: 实践是认识的来源、动力、目的和检验标准。项目中的每次火箭发射（实践）都为工程师们提供了新的数据和问题（认识的来源）。
2.  **认识反作用于实践**: 正确的认识能够指导实践，推动其发展。通过对失败数据的分析（理性认识），工程师们改进设计，指导了下一次的发射（实践）。
3.  **实践与认识的无限发展**: 从实践到认识，再从认识到实践，这一过程无限循环，使实践和认识都不断地向前发展，表现为一种波浪式前进和螺旋式上升的过程。本项目的可视化螺旋路径正是这一规律的直接体现。

## 📝 如何扩展

您可以轻松地添加自己的案例来展示“实践-认识”的辩证关系。

1.  **准备一张图片**: 为您的案例准备一张正方形的 `.png` 图标，并放入项目根目录。
2.  **修改 `app.js`**: 打开 `app.js` 文件，找到 `DATASETS` 常量。
3.  **添加新数据**: 仿照 `cz5` 和 `spacex` 的结构，添加一个新的键值对。一个基本的数据结构模板如下：

```javascript
const DATASETS = {
  // ... 已有数据
  
  your_key: { // 新案例的唯一标识符
    key: 'your_key',
    displayName: '新案例名称', // 用于切换按钮的显示
    headerTitle: '新案例：实践—认识—再实践的螺旋', // 右侧面板顶部的大标题
    docTitle: '实践与认识的辩-证运动：新案例的螺旋上升', // 浏览器标签页标题
    rocketSrc: 'your_image.png', // 你的图片文件名
    cycles: [
      {
        title: '阶段一标题',
        year: '年份范围',
        stages: [
          { id:'A', name:'实践-感性', summary:'阶段A的简短概括',
            detail:'阶段A的详细描述。这是实践的初始阶段，人们通过感官获得初步的、零散的经验。' },
          { id:'B', name:'理性认识', summary:'阶段B的简短概括',
            detail:'阶段B的详细描述。在感性认识的基础上，通过思考、分析、归纳，形成对事物本质的、系统的认识。' },
          { id:'C', name:'理性到实践的飞跃', summary:'阶段C的简短概括',
            detail:'阶段C的详细描述。将理性认识应用于实践，以检验和发展认识，并改造世界。' }
        ]
      },
      // ... 可以添加更多 cycle
    ]
  }
};
```

添加完毕后，刷新 `index.html` 页面，新的切换选项就会自动出现。
